# Interesting Surface

Interesting Surface is a mobile app that turns your screen into a playful canvas of particles that react to every touch. Use one finger or many at once to push, pull, and swirl particles across the display, and tweak parameters to shape the look and feel of the effect.

## Potential Features

- Multitouch particle interaction
- Adjustable particle count and size
- Custom color palettes
- Switch between effects (attract, repel, trails)
- Save and share presets
